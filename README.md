uniqle elements:

- "<ins>": represents a range of text added to a document. i used this to add some extra comments that didn't need to be there but was good for the reader to know. like a P.S. note. i used this in australia.html and southkorea.html.
- "<del>": represents text that has been "deleted" from a document. I used this as like an "edit" to something that maybe i shouldn't have put, so i put another phrase instead. used in southkorea.html.
- "<sup>": superscript. i used this to emphasize my first international trip of the year. used in elsalvador.html.

unique attributes:

- lang: used to identify a language in code. I used this to identify the Korean in southkorea.html for the places that I put.
- accesskey: used to hotkey a specific key to return to another page. used this in australia.html, southkorea.html, elsalvador.html.
- rel: used to describe the relationship between text and document. I used this in the footer sections, on the email to describe whose email it is that is on the footer. used in all pages.
